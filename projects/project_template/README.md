# Project Template

This scaffold is the canonical project layout for FilmCreator.

- duplicate `project_template` to a real slug when you start a project
- keep `04_references/` as the shared source of approved reusable refs
- keep clip-specific generation inside `05_scenes/<scene_id>/clips/<clip_id>/`
- keep authoring outputs in `02_story_analysis/` and `03_prompt_packages/`

Suggested first rename for a live project:

- `projects/project_template` -> `projects/<your_project_slug>`
