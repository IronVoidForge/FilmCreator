# Projects

Each film project should live under `projects/<project_slug>/`.

Use `projects/project_template/` as the starting point for a new project workspace. The intent is:

- shared approved references stay at the project level
- runtime scene work stays under `05_scenes/SC###/`
- clip-specific stills stay under `05_scenes/SC###/clips/CL###/stills/`
- state files track handoffs instead of relying on ad hoc filenames alone
